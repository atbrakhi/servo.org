Conference news

2023-06-19 After the project reactivation early this year, Servo project is getting some presence in different events.

In May 2023, Martin Robinson from the Servo project team had the chance to participate in RustNL 2023 conference in Amsterdam. There he gave a lighting talk about Servo project and the current plans. In addition, he participated in some follow up discussions about Servo and how some UI frameworks in the Rust ecosystem are exploring the possibility to use it.

In the first week of June 2023, Servo project team participated in the Web Engines Hackfest 2023. There, Delan Azabani gave a talk about Servo project, getting into details of the work performed in the first semester and the plans for the rest of the year; the talk provided some valuable insights on running demos on Servo, including updates on the status of floats and the transition of Servoâ€™s layout. Also, there were some engaging discussions around the future of the Stylo crate in Gecko and Servo.

Apart from that in the same event there was a breakout session about Servo where various topics were addressed, such as webrender, style crate, and the synchronization process for both their current and future developments. Of particular interest was the implementation of the Embedding API and the associated challenges. Lastly, there was a discussion on simplifying the contribution process to the Servo project, with a focus on making it more beginner-friendly, enabling a broader participation from interested individuals. You can check the session notes online.

End of the June, Servo is going to be present at Embedded Open Source Summit in Prague. If you are participating in such event, feel free to approach us and ask any question about the project status and future plans.
